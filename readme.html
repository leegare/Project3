Introduction

This repository contains the following folders and files:

  • Data>Raw: contains the csv files downloaded from the kaggle project (hour.csv and day.csv) as well as 4 csv files containing the data of each quarter of the year 2017 from the bike rental company in Washington D.C. Finally a sqlite file with the hour and day csv files.

    Data>Interim: Has the sqlite file with the hour and day csv files.

    Data>Processed: Has 3 csv files of the data pre-processed and downsampled by the hour, day and week.

  • Reports>Img: Graphs and charts used to illustrate the project.

  • Notebooks:


    • 1_Acquire_refine_explore: Python script to read the csv files and preprocess them.

    • Q1: python script dealing with question 1

    • Q2: python script dealing with question 2

    • Q3: python script dealing with question 3

    • Q4: python script dealing with question 4

    • Q5: python script dealing with question 5

    • Notebook: python script dealing with all the notebooks described above

    • Project: Markdown script briefing the whole project.
